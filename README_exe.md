# はじめに

この実行ファイルでは、ユドナリウムコネクト.wiki ( <https://udonarium-connect.wiki/> ) 用のシャドウバウトテーブルを作成することができます。
初期状態では仮画像のみが含まれており、ビルドを行うと仮画像で遊ぶzipファイルが出来上がります。
アイドルマスターミリオンライブシアターデイズのX公式アカウントアカウントで配布されているpdfを実行ファイルと同じフォルダに配置することで公式の画像で遊ぶファイルが出来上がります。

## 注意事項

ShadowboutBuilderフォルダ、およびビルドされたzipファイルについて、公式の画像が含まれているものは再配布を禁止いたします。また配布したことによって発生したすべての問題についての一切の責任は負いません。
このフォルダに含まれている実行ファイルを実行した段階で上記の禁止事項と責任事項についての同意をしたものと判断いたします。

## 動作環境

- Windows11 バージョン25H2以降

## このフォルダについて

このフォルダには、以下の実行ファイルが含まれています。

- build.exe
  - ユドナリウムコネクト.wikiで遊ぶためのzipファイルを作成するための実行ファイルです。

- prepare.exe
  - 公式画像をPDFから切り出すための実行ファイルです。

また仮画像のフォルダとビルドに必要なデータが含まれています。

## 使い方

### 仮画像でそのまま遊ぶ場合

- build.exeを実行してください。
- 出来上がったShadowbout.zipをユドナリウムコネクト.wikiのメイン画面から、ZIP読込を選択してください。
- 出来上がったテーブルで公式ルールに従い遊んでください。

### 公式の配布している画像で遊ぶ場合

#### PDFの配置

- アイドルマスターミリオンライブシアターデイズの公式Xアカウントの投稿より、シャドウバウトの配布PDFをダウンロードしてください
  - 公式アカウントURL: <https://x.com/imasml_theater>
  - 2026/4/27 現在の配布URL: <https://x.com/imasml_theater/status/2047601441863282705>
    - 上記URLは場合によっては投稿が削除されるなどして機能しなくなる場合があります。
- ダウンロードしたPDFをprepare.exe と同じフォルダに配置してください。

#### 画像切り出し実行

- prepare.exeを実行してください。コンソール画面が立ち上がります。
- 暫く待つと、コンソール画面が消えます。
- Cardフォルダに画像が切り出されていることが確認できれば完了しています。

#### ZIPファイルの準備

- build.exeを実行してください。
- 出来上がったShadowbout.zipをユドナリウムコネクト.wikiのメイン画面から、ZIP読込を選択してください。
- 出来上がったテーブルで公式ルールに従い遊んでください。
