# はじめに

このスクリプトでは、ユドナリウムコネクト.wiki ( <https://udonarium-connect.wiki/> ) 用のシャドウバウトテーブルを作成することができます。
初期状態では仮画像のみが含まれており、ビルドを行うと仮画像で遊ぶzipファイルが出来上がります。
アイドルマスターミリオンライブシアターデイズのX公式アカウントアカウントで配布されているpdfをスクリプトと同じフォルダに配置することで公式の画像で遊ぶファイルが出来上がります。

## 注意事項

ShadowboutBuilderフォルダ、およびビルドされたzipファイルについて、公式の画像が含まれているものは再配布を禁止いたします。また配布したことによって発生したすべての問題についての一切の責任は負いません。
このフォルダに含まれているスクリプトを実行した段階で上記の禁止事項と責任事項についての同意をしたものと判断いたします。

## 動作環境

- Windows11 バージョン25H2以降
- Utils以下にあるPythonファイルを実行する場合はPython3系であれば動くはずです
  - 作者環境では3.10系で動くことを確認しています

## このフォルダについて

このフォルダには、以下のバッチスクリプトが含まれています。

- build.bat
  - ユドナリウムコネクト.wikiで遊ぶためのzipファイルを作成するためのスクリプトです。

- prepare.bat
  - 公式画像をPDFから切り出すためのスクリプトです。

- clear.bat
  - 再配布可能な形になるように、リセットをするスクリプトです。

Utilsフォルダ以下には同等の動作をするpythonファイルが含まれています

- Utils/build.py
  - build.batと同等の動作をします

- Utils/prepare.py
  - prepare.batと同等の動作をします

また仮画像のフォルダとビルドに必要なデータが含まれています。

## 使い方(exeファイル)

- Releaseページにwindows実行用ファイルのzipがリリースされていますのでダウンロードしてください。
- zipファイルを解凍してください。
- 仮画像のままでOKであればbuild.exeを実行してください。
- exeファイルと同じ位置にpdfを用意すればprepare.exeが切り出しを実行してくれます。その後build.exeでzipを作成してください。
- 出来上がったShadowbout.zipをユドナリウムコネクト.wikiのメイン画面から、ZIP読込を選択してください。
- 出来上がったテーブルで公式ルールに従い遊んでください。

## 使い方(batファイル)

### 仮画像でそのまま遊ぶ場合

- build.batを実行してください。
- 出来上がったShadowbout.zipをユドナリウムコネクト.wikiのメイン画面から、ZIP読込を選択してください。
- 出来上がったテーブルで公式ルールに従い遊んでください。

### 公式の配布している画像で遊ぶ場合

#### PDFの配置

- アイドルマスターミリオンライブシアターデイズの公式Xアカウントの投稿より、シャドウバウトの配布PDFをダウンロードしてください
  - 公式アカウントURL: <https://x.com/imasml_theater>
  - 2026/5/11 現在の配布URL: <https://x.com/imasml_theater/status/2047601441863282705>
    - 上記URLは場合によっては投稿が削除されるなどして機能しなくなる場合があります。
- ダウンロードしたPDFをREADMEと同じフォルダに配置してください。

#### ImageMagickとGhostScriptのダウンロード

- ダウンロードしたPDFから画像を切り出すため、ImageMagickとGhostScriptをダウンロード、インストールしてください。
- Powershellを立ち上げ、magickコマンドが実行できることと、magickのバージョン情報 (`magick -version` で確認できます) に含まれるDelegatesにgslibが含まれていることを確認してください。
  - magickコマンドが実行できない場合 ... ImageMagickのインストールに失敗しています。
  - gslibがない場合 ... GhostScriptのインストールに失敗しているか、ImageMagickがGhostScriptを認識していません。

#### 画像切り出し実行

- prepare.batを実行してください。コンソール画面が立ち上がります。
- 暫く待つと、コンソール画面が消えます。
- Cardフォルダに画像が切り出されていることが確認できれば完了しています。

#### ZIPファイルの準備

- build.batを実行してください。
- 出来上がったShadowbout.zipをユドナリウムコネクト.wikiのメイン画面から、ZIP読込を選択してください。
- 出来上がったテーブルで公式ルールに従い遊んでください。

### 　リセットする場合

- clear.batを実行してください。
- このフォルダから公式の画像が消去されます。
- 念のため、CardフォルダとBackgroundフォルダに画像が含まれていないかを確認してください。
